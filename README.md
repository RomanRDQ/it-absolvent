# My result of IT Absolvent
 In this project I would like to represent some projects on which I was practicing my skils
##  Stack:
- React,js
- Node.js
- TypeScript
- Material UI, styled-components


## Installation:
```sh
git clone https://github.com/RomanRDQ/it-absolvent.git
npm i
npm run start
```
